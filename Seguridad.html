Protocolos de Seguridad IT

Normativas de contraseñas y configuración de Firewalls

1. Política de Contraseñas

Directrices obligatorias para todos los usuarios del sistema:

  • Longitud: Mínimo 12 caracteres.
  • Complejidad: Uso de mayúsculas, minúsculas, números y símbolos.
  • Rotación: Cambio cada 90 días naturales.
  • MFA: Autenticación de doble factor obligatoria para acceso VPN.

2. Configuración de Firewalls

Reglas de filtrado perimetral establecidas:

Acción Puerto Protocolo
ALLOW 443 HTTPS (Web Segura)
DENY 21 FTP (Inseguro)
DROP 3389 RDP (Escritorio Remoto)
