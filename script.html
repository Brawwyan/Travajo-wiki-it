Automatización de Copias de Seguridad con PowerShell

Script completo

# Ruta origen
$origen = "C:\Datos"

# Ruta destino
$destino = "D:\Backups"

# Fecha actual
$fecha = Get-Date -Format "yyyy-MM-dd_HH-mm"

# Crear carpeta con fecha
$rutaBackup = "$destino\Backup_$fecha"
New-Item -ItemType Directory -Path $rutaBackup

# Copiar archivos
Copy-Item -Path $origen -Destination $rutaBackup -Recurse

# Eliminar backups antiguos (más de 7 días)
Get-ChildItem $destino | Where-Object {
  $_.CreationTime -lt (Get-Date).AddDays(-7)
} | Remove-Item -Recurse -Force

Explicación línea por línea

  • $origen: Define la carpeta que se quiere copiar.
  • $destino: Define dónde se guardarán las copias de seguridad.
  • Get-Date: Obtiene la fecha actual para crear backups únicos.
  • $rutaBackup: Construye el nombre de la carpeta con fecha.
  • New-Item: Crea la carpeta de backup.
  • Copy-Item: Copia todos los archivos y carpetas de forma recursiva.
  • Get-ChildItem: Lista los backups existentes.
  • Where-Object: Filtra los backups antiguos (más de 7 días).
  • Remove-Item: Elimina automáticamente los backups antiguos.
