3. Configurar DHCP

  1. Abrir herramientas DHCP
  2. Crear un nuevo ámbito
  3. Definir nombre del ámbito

Ejemplo de rango IP

Parámetro Valor
Red192.168.1.0
Máscara255.255.255.0
Inicio192.168.1.100
Fin192.168.1.200
Gateway192.168.1.1

4. Configurar opciones DHCP

  • Puerta de enlace predeterminada
  • Servidor DNS
  • Dominio
Opción Ejemplo
Router192.168.1.1
DNS8.8.8.8
Dominioempresa.local

5. Activar el ámbito

  1. Ir al panel DHCP
  2. Seleccionar el ámbito creado
  3. Hacer clic en "Activar"

6. Verificación

  • Conectar un cliente a la red
  • Ejecutar ipconfig /renew
  • Comprobar IP asignada
